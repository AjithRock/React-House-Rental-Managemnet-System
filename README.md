## Setup 
1. Fork the repo and then clone it or download it.

2. First install all dependencies:
    ```bash
    # with npm
    npm install

    # or with yarn
    yarn
    ```

5. Start the server
    ```javascript
    npm start [OR] yarn start
    ```

6. Now run the web app
    ```javacript
    localhost:[PORT] (PORT=defined in .env)
    ```
